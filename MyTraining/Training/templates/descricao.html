Descrição do Sistema

O projeto "MyTraining" busca criar um Personal Trainer sem a necessidade de um atendimento longo com um profissional humano. Durante a primeira fase de implantação do sistema, o mesmo deverá ser utilizado como ferramenta de apoio a funcionários de qualquer academias.

Enquanto o sistema sugere treinos para os clientes, o profissional registra os treinos como adequados ou não. Caso o treino sugerido não seja adequado ao cliente, o profissional pode informar qual seria o treino ideal, "ensinando" o sistema.

Este aprendizado ocorre por meio da inclusão de amostras no dataset de aprendizado. Este dataset é registrado e mantido no banco de dados, garantindo que o sistema não perca conhecimento ao ser desligado. O intuito é que, com o passar do tempo, a rede possa sugerir treinos com ótima precisão.
